Analyzing Political Ideology Using NLP on Public Statements

NOTE: This project focuses on the technical application of NLP for political discourse analysis and is intended to be nonpartisan in nature.

Project Inspiration: I was interested in this given the recent presidential election, where it seemed like there is very little accountability for the statements politicians make to specific groups of individuals to gain their support that does not necessarily align with the politician's stated political platform. My intent was to build a lightweight, fast method to analyze how a politician's rhetoric ranks along the political spectrum.

Project Details: To address the lack of accountability in political rhetoric, I developed a data-driven NLP framework to evaluate how closely a politician's statements align with their stated political platform. Using Natural Language Processing (NLP) techniques, I analyzed over 37,000 sentences from VoteSmart.org, including speech transcripts, tweets, and interviews. I then ranked politicians' ideological alignment by comparing their rhetoric to the Institute of Legislative Analysis' Congressional Limited Government Scorecard.

    • SpaCy
    • DistilBERT
    • POS Tagging

Lung Cancer Detection in Histopathological Images

Project Inspiration: During a routine dental check-up, I noticed my hygienist using an AI-assisted tool to analyze X-rays, highlighting areas of concern. This sparked my curiosity about using neural networks for medical imaging. At the same time, I was exploring transformers and wanted to see how attention mechanisms could improve image classification models.

Project Details: Inspired by AI-assisted dental imaging, I explored deep learning models for medical diagnostics. I developed and compared a Convolutional Neural Network (CNN) and a Vision Transformer (ViT) for lung cancer classification in histopathological images. The ViT model, leveraging attention mechanisms, outperformed the CNN by reducing false negatives and enhancing interpretability, providing critical visual insights for medical professionals.

  • Programming: Python
  • Deep Learning: CNNs, Vision Transformers (ViT)
  • Libraries & Tools: TensorFlow, Keras, Hugging Face Transfomers
  • Data Processing: Histopathological image resizing (224x224), normalization, dataset splitting (70-10-20 train-validation-test split)
  • Visualization: ViT attention maps for model interpretability, precision-recall heatmaps, and comparative bar charts for performance metrics

Detecting Illegal Fishing

Project Inspiration: My time in the military included extended periods at sea, where we frequently encountered unidentified vessels approaching, often with unclear intentions. This experience made me aware of the challenges in maritime monitoring and the importance of detecting suspicious activity early. Illegal, unreported, and unregulated (IUU) fishing is a major issue, with an estimated 75% of illegal fishing going undetected, leading to significant environmental and economic damage. This project aims to leverage machine learning to enhance maritime surveillance and improve the detection of illegal fishing activity.

Project Description: This project aims to identify fishing activity in trawling vessels to combat illegal, unreported, and unregulated (IUU) fishing, which harms marine ecosystems and depletes fish stocks. Using data from Global Fishing Watch, patterns in vessel behavior, such as speed, location, and distance from shore, were analyzed to detect when vessels were actively fishing. A Random Forest model performed best, achieving 88% accuracy by incorporating information about recent vessel movements. The analysis revealed that fishing vessels typically move slower and operate in specific ocean regions, distinguishing them from vessels in transit or docked. However, the complexity of vessel behavior led to occasional misclassifications, potentially overwhelming monitoring systems. To improve results, future efforts should incorporate more advanced methods and additional factors influencing fishing activity. This study highlights the potential of machine learning to monitor global fishing practices and protect marine ecosystems.

  • Programming: Python
  • Models Used: K-Means Clustering, DBSCAN (Density-Based Spatial Clustering), Random Forest (with and without lag variables), Linear SVM (Support Vector Machine), Polynomial SVM, Radial Basis Function (RBF) SVM
  • Libraries & Tools: Python, pandas, numpy, matplotlib, seaborn, scikit-learn, GeoPandas, shapely, Jupyter Notebook
  • Data Processing: Histopathological image resizing (224x224), normalization, dataset splitting (70-10-20 train-validation-test split)
  • Visualization: ViT attention maps for model interpretability, precision-recall heatmaps, and comparative bar charts for performance metrics

Sentiment Analysis of Yelp Reviews

This project applies Naive Bayes and Decision Tree classifiers to analyze Yelp restaurant reviews, focusing on two tasks: identifying positive and negative sentiments and detecting fake reviews. For the sentiment classification task, Naive Bayes models achieved up to 68.5% accuracy, effectively distinguishing between positive and negative reviews. In contrast, for the more challenging task of detecting fake reviews, Decision Tree classifiers performed slightly better than Naive Bayes, with an accuracy of 35.9%, highlighting the significant challenges in reliably identifying deceptive content.

  • Algorithms: MNB, BNB, Decision Tree
  • Libraries & Tools: NLTK, Scikit-Learn, pandas, numpy, matplotlib, wordcloud

Detecting patterns in news coverage of political figures

This project applies K-Nearest Neighbors (KNN) and Hierarchical Clustering with Cosine Similarity to classify news articles and detect patterns in media coverage of political figures. While KNN focused on supervised classification to identify articles by their subject, Hierarchical Clustering highlighted significant overlaps in language use across topics, achieving limited success in distinguishing between figures like Trump, Harris, Walz, and Vance due to shared media narratives.

  • Algorithms: KNN, HAC
  • Libraries & Tools: Scikit-Learn, pandas, numpy, matplotlib, seaborn, wordcloud, NewsAPI, scipy
