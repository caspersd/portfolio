Detecting Illegal Fishing

This project aims to identify fishing activity in trawling vessels to combat illegal, unreported, and unregulated (IUU) fishing, which harms marine ecosystems and depletes fish stocks. Using data from Global Fishing Watch, patterns in vessel behavior, such as speed, location, and distance from shore, were analyzed to detect when vessels were actively fishing. A Random Forest model performed best, achieving 88% accuracy by incorporating information about recent vessel movements. The analysis revealed that fishing vessels typically move slower and operate in specific ocean regions, distinguishing them from vessels in transit or docked. However, the complexity of vessel behavior led to occasional misclassifications, potentially overwhelming monitoring systems. To improve results, future efforts should incorporate more advanced methods and additional factors influencing fishing activity. This study highlights the potential of machine learning to monitor global fishing practices and protect marine ecosystems.

  • Programming: Python
  • Models Used: K-Means Clustering, DBSCAN (Density-Based Spatial Clustering), Random Forest (with and without lag variables), Linear SVM (Support Vector Machine), Polynomial SVM, Radial Basis Function (RBF) SVM
  • Libraries & Tools: Python, pandas, numpy, matplotlib, seaborn, scikit-learn, GeoPandas, shapely, Jupyter Notebook
  • Data Processing: Histopathological image resizing (224x224), normalization, dataset splitting (70-10-20 train-validation-test split)
  • Visualization: ViT attention maps for model interpretability, precision-recall heatmaps, and comparative bar charts for performance metrics

Sentiment Analysis: Classifying Tweets by Emotions

This project aims to analyze textual data labeled with emotions to uncover patterns, classify text by emotional content, and explore the nuanced interplay between various emotional expressions. By leveraging machine learning techniques such as Naive Bayes, Support Vector Machines (SVM), Decision Trees, and Latent Dirichlet Allocation (LDA), the study seeks to accurately identify emotions and reveal latent themes in the data. The findings have practical applications in fields like customer sentiment analysis, healthcare, and public policy, enabling organizations to better understand emotional tones and respond effectively.

  • Algorithms: Naive Bayes (Multinomial, Bernoulli), Latent Dirichlet Allocation (LDA), Decision Tree (Unpruned, Pruned), SVM (Linear, Polynomial, RBF Kernels), MiniBatchKMeans (Clustering)
  • Libraries & Tools: Scikit-learn, NLTK, Pandas, NumPy, LDAvis
  • Data Processing: Count Vectorizer, TF-IDF Transformer, Principal Component Analysis (PCA)
  • Visualization: Matplotlib, Seaborn, WordCloud, LDAvis

Naive Bayes (Multinomial and Bernoulli), Decision Tree

This project applies Naive Bayes and Decision Tree classifiers to analyze Yelp restaurant reviews, focusing on two tasks: identifying positive and negative sentiments and detecting fake reviews. For the sentiment classification task, Naive Bayes models achieved up to 68.5% accuracy, effectively distinguishing between positive and negative reviews. In contrast, for the more challenging task of detecting fake reviews, Decision Tree classifiers performed slightly better than Naive Bayes, with an accuracy of 35.9%, highlighting the significant challenges in reliably identifying deceptive content.

K-Mean Clustering and Hierarchical Clustering (with Cosine Similarity)

This project applies K-Nearest Neighbors (KNN) and Hierarchical Clustering with Cosine Similarity to classify news articles and detect patterns in media coverage of political figures. While KNN focused on supervised classification to identify articles by their subject, Hierarchical Clustering highlighted significant overlaps in language use across topics, achieving limited success in distinguishing between figures like Trump, Harris, Walz, and Vance due to shared media narratives.
